# S3-mobile — Android Share-Sheet Capture — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Register the burkmak Flutter app as an Android **share target** so sharing a URL from any app opens a quick-save that calls the existing `POST /items` and reports success — including the signed-out case (sign in, then the held URL saves).

**Architecture:** The app is a lib-only Dart package today; `flutter create` adds `android/` + `ios/`. `receive_sharing_intent` surfaces shared text (cold-start + warm-stream). A pure `extractFirstUrl` pulls the URL out; a thin `ShareIntentService` routes it: if a bearer token exists, navigate (via a global navigator key) to a `QuickSaveScreen` that **reuses the existing `AddLinkCubit`** and auto-fires the save; if signed out, the URL is held in a `PendingShareStore` and consumed by `AuthGate` once authenticated. iOS folder is scaffolded but its Share Extension is deferred.

**Tech Stack:** Flutter 3.41 / Dart 3.8, `flutter_bloc`, `get_it`, Dio, `slang` (en/el/ru/uk), `receive_sharing_intent` (new), `bloc_test` + `mocktail`. Package name **`app_mobile`** (imports `package:app_mobile/...`). Flutter binaries at `~/projects/petProjects/tools/flutter/bin` (not on PATH).

**Branch:** `feat/s3-mobile` off `main`. Commit per task. `--no-ff` merge to `main` when gates pass.

**Verification ceiling (be honest):** `flutter analyze` + `flutter test` cover the Dart. OS share registration (manifest intent-filter, real share-sheet entry) is **device-verifiable only** — out of this sandbox.

// apps/mobile/lib/shared/share/share_url_parser.dart

final _urlPattern = RegExp(r'https?://[^\s]+', caseSensitive: false);

/// Returns the first http(s) URL found in shared text, or null if none.
/// Shared content can be a bare URL or a sentence containing one.
String? extractFirstUrl(String? text) {
  if (text == null) return null;
  final match = _urlPattern.firstMatch(text.trim());
  return match?.group(0);
}
```

// apps/mobile/lib/shared/share/pending_share_store.dart

/// Holds a shared URL captured while the user was signed out, so it can be
/// saved once they authenticate. Single-slot: a newer share replaces an older.
class PendingShareStore {
  String? _url;

  void set(String url) => _url = url;

  /// Returns the held URL and clears it (consume-once).
  String? take() {
    final url = _url;
    _url = null;
    return url;
  }
}
```

- [ ] **Step 1: Write the service**

```dart
// apps/mobile/lib/shared/share/share_intent_service.dart
import 'dart:async';

import 'package:receive_sharing_intent/receive_sharing_intent.dart';

import 'package:app_mobile/app/routes.dart';
import 'package:app_mobile/shared/auth/token_storage.dart';
import 'package:app_mobile/shared/navigation/app_navigator.dart';
import 'package:app_mobile/shared/share/pending_share_store.dart';
import 'package:app_mobile/shared/share/share_url_parser.dart';

/// Listens for URLs shared into the app and routes them to the quick-save flow.
/// Signed in → push quick-save. Signed out → hold the URL for AuthGate.
class ShareIntentService {
  ShareIntentService(this._tokens, this._pending);

  final TokenStorage _tokens;
  final PendingShareStore _pending;
  StreamSubscription<List<SharedMediaFile>>? _sub;

  /// Call once after the first frame (navigator must exist).
  Future<void> start() async {
    final initial = await ReceiveSharingIntent.instance.getInitialMedia();
    _handle(initial);
    await ReceiveSharingIntent.instance.reset();

    _sub = ReceiveSharingIntent.instance.getMediaStream().listen(_handle);
  }

  Future<void> dispose() async {
    await _sub?.cancel();
  }

  Future<void> _handle(List<SharedMediaFile> media) async {
    if (media.isEmpty) return;
    final text = media
        .map((m) => m.path)
        .firstWhere((p) => extractFirstUrl(p) != null, orElse: () => '');
    final url = extractFirstUrl(text);
    if (url == null) return;

    final token = await _tokens.read();
    final signedIn = token != null && token.isNotEmpty;
    if (signedIn) {
      appNavigatorKey.currentState?.pushNamed(AppRoutes.quickSave, arguments: url);
    } else {
      _pending.set(url);
    }
  }
}
```

> If `receive_sharing_intent`'s API names differ in the resolved version (e.g. `getInitialMedia`/`getMediaStream`/`SharedMediaFile.path`), adapt to the installed package's README — the routing logic stays identical. Shared plain text arrives as `SharedMediaFile.path` for `SharedMediaType.text` in current versions.
